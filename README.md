# 云峰智谷简历生成系统

这是一个基于Web的简历生成系统，帮助用户创建自主招生简历。

## 功能特点

- 手机号注册/登录
- 验证码登录
- 简历编辑与生成
- 响应式设计

## 技术栈

- HTML5
- CSS3
- JavaScript

## 页面预览

- 登录/注册页面：提供手机号和验证码登录功能
